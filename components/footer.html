💡 Về dự án HCM202

🎯 Mục tiêu

Tạo nền tảng học tập hiện đại, tương tác về tư tưởng Hồ Chí Minh, giúp sinh viên hiểu sâu và áp dụng các giá trị đạo đức cách mạng vào cuộc sống.

⚡ Công nghệ

HTML5, CSS3, Vanilla JavaScript, Responsive Design, Progressive Web App principles, RESTful API integration.

📊 Thống kê

6 trang chính • 50+ trích dẫn • 20+ tình huống thực tế • AI Chatbot integration • Quiz tương tác • LocalStorage để lưu cam kết

🤖 AI & Công cụ hỗ trợ

GitHub Copilot MCP Server (hướng dẫn code) • Coze AI (chatbot tương tác) • Perplexity (research & tra cứu) • ChatGPT (chuẩn hóa dữ liệu) • NotebookLM (định hướng theo giáo trình)

🏆 Giá trị đạt được

Nâng cao hiểu biết về tư tưởng HCM • Phát triển tư duy phản biện • Rèn luyện kỹ năng frontend • Teamwork và project management
